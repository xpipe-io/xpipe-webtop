# XPipe Webtop

XPipe Webtop is a web-based desktop environment that can be run in a docker/OCI container and accessed from a browser window.
The desktop environment comes with XPipe and various terminals, editors, and other useful tools preinstalled and configured.

You can find the documentation at [https://docs.xpipe.io/guide/webtop](https://docs.xpipe.io/guide/webtop).
